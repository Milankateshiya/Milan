Are the HTML tags and elements the same thing ?

HTML Tags
-The starting and ending point parts of an HTML document are HTML Tags, which start with < (less than) and end with >(greater than) angle bracket.
whatever is written inside the angle brackets are known as tags.

They are keyowrds where every specific tag has some unique meaning.
HTML tags can also contain attributes, which are used to provide additional information about the element. Attributes are defined inside the opening tag and are separated by spaces.
The value of the attribute is defined in quotes.

Paragraph-<p> </p>
