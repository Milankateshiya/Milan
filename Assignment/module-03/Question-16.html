how to active links different from normal links?

In an link is a hyperlink that is not yet clicked by the user.
the unvisited links will be in blue in color with an underline

visited link is a hyperlink that is clicked by the user.
These indicate that the links have been visited previously.

Active link is a hyperlink that is currently being interacted with the user.
user holds the mouse button on the link and not released yet or if right clicked on it, it will change its color into red.
