What is the rule set?

ans.

A selector list and an associated declarations block, together, are called a ruleset, or often a rule.

Each (valid) declaration block is preceded by one or more comma-separated selectors, which are conditions selecting some elements of the page.

A CSS ruleset is various affirmations to various pieces or elements of the document.
The objective is to apply a bunch of properties for certain distinct qualities to a solitary, or a particular arrangement of components in the connected HTML page.

The beginning indicates that the rule created will be a class, also “container” indicates the name of the selector.

which contains some CSS properties and their corresponding values.
The CSS ruleset will be applied when the selector name is called on the main HTML page.
